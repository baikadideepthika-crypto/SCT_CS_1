# Caesar Cipher - SkillCraft Technology Internship

## Task 1 - Cyber Security Internship

A Python program that encrypts and decrypts text using the Caesar Cipher algorithm.

## Features
- Encrypt any text message
- Decrypt any encrypted message
- Supports uppercase and lowercase letters
- Non-letter characters (spaces, numbers, symbols) remain unchanged
- Input validation for shift value (1-25)

## How to Run
```bash
py caesar_cipher.py
```

## How It Works
Caesar Cipher works by shifting each letter in the message by a fixed number of positions in the alphabet.

Example:
- Message: `Hello World`
- Shift: `3`
- Encrypted: `Khoor Zruog`
